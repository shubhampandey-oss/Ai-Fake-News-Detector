About This System

Understanding the technology behind fake news detection

What is this system?

An early-stage fake news detection and credibility assessment system using linguistic analysis, semantic modeling, source reputation scoring, and multi-source evidence aggregation.

Important: This system performs probabilistic credibility assessment, NOT absolute truth verification. Results should be interpreted as "likelihood of credibility" based on available evidence.

Key Features

ML-Based Classification

Multiple models (Logistic Regression, SVM, Random Forest, LSTM) trained locally to predict fake vs real news.

Linguistic Analysis

Examines writing style, readability, sensational language use, and vocabulary patterns.

Source Credibility

Pre-compiled database of source reputation scores based on public credibility reports.

Evidence Aggregation

Combines multiple signals (model, source, evidence) for robust credibility scoring.

Technology Stack

Python NumPy Pandas scikit-learn PyTorch NLTK Gensim Flask Bootstrap

All models are trained locally from scratch. No external AI APIs or pretrained models are used.

Limitations

  • Cannot verify facts against the internet in real-time
  • Source credibility is based on pre-compiled data, not dynamic assessment
  • Performance depends on training data quality and domain coverage
  • May not detect sophisticated disinformation campaigns
  • Should be used as one tool among many, not as sole arbiter of truth

Research Context

This system is designed as a B.Tech final-year / IEEE-style research project demonstrating practical application of NLP and machine learning techniques for media credibility assessment.
